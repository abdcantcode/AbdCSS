# AbdCSS — Style Engine

AbdCSS is a browser-native style engine with zero dependencies. It comes with predefined `chai-` utility classes and lets you define your own on the fly. Write HTML, apply classes, see it render live. Custom rules save to localStorage and export as clean JSON.

## Tech Stack

- HTML
- CSS
- JavaScript

## Features

- Predefined `chai-` utility classes covering layout, spacing, typography, color, and more
- Define custom rules on the fly — name a class, pick a CSS property, set a value
- Live HTML preview that updates as you type
- Custom rules persist via localStorage
- Clean JSON export of all custom rules
- Zero dependencies — runs in a single HTML file
